## Git Basics
Git is a version control system that helps track changes in files and coordinate work on those files among multiple people. Here are some fundamental commands:
1. `git init`: Initializes a new Git repository.
2. `git clone <repository-url>`: Clones an existing repository.
3. `git add <file>`: Stages changes for commit.
4. `git commit -m "commit message"`: Commits staged changes with a message.
5. `git push`: Pushes changes to a remote repository.

## Best Practices
- Commit often with clear messages.
- Write meaningful commit messages to describe your changes.
- Use branches to manage new features or fixes safely.
- Keep your repository organized and clean.
